# CesiumWebpackVueCli

> A CesiumWebpackVueCli

## Build Setup

``` bash
# 安装依赖
npm install

# 安装cesium模块
npm install cesium 

# 运行开发环境
npm run dev

# 编译
npm run build

# 编译分析报告
npm run build --report

# 打开浏览器查看运行结果
localhost:8080
```
